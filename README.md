This project is a Machine Learning-based web application built using Streamlit.
It allows users to input data and get real-time predictions using trained models 
such as Logistic Regression, Decision Tree Classifier, and Random Forest Classifier.
The models are trained on a dataset and saved using Joblib for reuse.
The application provides an interactive and user-friendly interface where predictions are displayed instantly
based on user inputs.
The project demonstrates the complete workflow of a machine learning system, 
from data preprocessing and model training to deployment as a web application.
